## 怎样构建一个React程序
1. 先画出web的草图
2. 将图划分为不同的组件
3. 实现静态版本的程序和组件
4. 将静态版本组合起来
5. 考虑state的组成和实现
6. 添加交互方法
7. 将这些组合在一起，完成最终的版本
